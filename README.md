# Giới thiệu
Project này là một vài ví dụ về [Puppeteer](https://github.com/GoogleChrome/puppeteer) - Một thư viện để điều khiển Chromium hay Chrome qua DevTool

# Chạy project
- Sử dụng lệnh `cd` để truy cập vào folder sample
- Chạy file `index.js`
